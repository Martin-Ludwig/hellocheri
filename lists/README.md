# Lists

A lists application built with React (frontend) and Elysia (backend).

## Structure

```
lists/
├── packages/
│   ├── shared/    # Types, interfaces, validation
│   ├── frontend/  # React App (Bun bundled)
│   └── backend/   # Elysia REST API
├── package.json   # Workspace root
└── bunfig.toml
```

## Getting Started

```sh
bun install
bun dev
```
